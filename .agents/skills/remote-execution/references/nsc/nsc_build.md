## nsc build

Build an image in a build instance.

```
nsc build [flags]
```

### Options

```
      --build-arg strings     Pass build arguments to the build.
  -f, --file string           If set, specifies what Dockerfile to build.
  -h, --help                  help for build
      --load                  If specified, load the image to the local docker registry.
  -n, --name strings          Provide a list of name tags for the image in nscr.io Workspace registry
      --output-local string   If set, outputs the build results to the specified directory.
      --platform strings      Set target platform for build.
  -p, --push                  If specified, pushes the image to the target repository.
      --secret stringArray    Secret to expose to the build (format: "id=mysecret[,src=/local/secret]")
  -t, --tag strings           Attach the specified tags to the image.
```
