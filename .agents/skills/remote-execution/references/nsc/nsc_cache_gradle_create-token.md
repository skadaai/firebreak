## nsc cache gradle create-token

Create a revokable token for accessing the Gradle cache.

```
nsc cache gradle create-token [flags]
```

### Options

```
      --cache_name string     Select a Gradle cache to grant access to. By default, all Gradle caches can be accessed.
      --expires_in duration   Duration until the token expires. (default 2160h0m0s)
  -h, --help                  help for create-token
      --scope tenant          Set the scope of the generated access token. Valid options: tenant, `user`. Tokens with user scope are bound to the tenant membership of the current user. (default "user")
      --token string          Write token to this file in JSON format. (default "token.json")
```
