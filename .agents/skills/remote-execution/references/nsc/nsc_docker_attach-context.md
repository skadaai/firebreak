## nsc docker attach-context

Configures a Docker context that uses an ephemeral environment.

```
nsc docker attach-context [flags]
```

### Options

      --machine_type string   Specify the machine type.
      --name string           The name of the context that is created; by default it will be nsc-{id}.
      --new                   If set, creates a new instance.
      --state string          If set, stores the proxy socket in this directory.
      --to string             Attaches a context to the specified instance.
