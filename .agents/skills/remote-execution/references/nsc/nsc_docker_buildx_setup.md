## nsc docker buildx setup

Setup buildx in the current machine, to use Namespace Remote builders.

```
nsc docker buildx setup [flags]
```

### Options

```
      --background          If true, runs the remote builder context in the background.
      --create_at_startup   If true, creates the build clusters eagerly.
      --force_cleanup       If set, it forces a cleanup of any previous buildx proxy running in background.
  -h, --help                help for setup
      --name string         The name of the builder we setup. (default "nsc-remote")
      --state string        If set, stores the remote builder context details in this directory.
      --tag string          If set, target a specific remote builder.
      --use                 If true, changes the current builder to nsc-remote.
```
