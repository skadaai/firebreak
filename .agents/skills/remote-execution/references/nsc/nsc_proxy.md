## nsc proxy

Provides proxy support for instance services.

```
nsc proxy [instance-id] [flags]
```

### Options

```
  -h, --help               help for proxy
      --once               If set, stop the proxy after a single connection.
  -o, --output string      One of plain or json. (default "plain")
      --output_to string   If specified, write the JSON configuration to this path.
  -s, --service string     The service to proxy (e.g. vnc, rdp).
      --websocket          If set, outputs information to connect to the service directly using websockets.
```
