## nsc registry update-image-expiration

Update the expiration time of an image.

### Synopsis

Update the image expiration of the specified image.

The image reference can be provided as a positional argument in the format repository@digest (e.g., myrepo@sha256:abc123...)

Alternatively, use --repository and --digest flags.

```
nsc registry update-image-expiration [image-reference] [flags]
```

### Options
