## nsc run

Starts a container in an ephemeral environment, optionally exporting ports for public serving.

```
nsc run [flags]
```

      --features strings         A set of features to attach to the instance.

      --user string              Customize the user to run the container as.
      --volume strings           Attach a volume to the instance, {cache|persistent}:{tag}:{mountpoint}:{size}
