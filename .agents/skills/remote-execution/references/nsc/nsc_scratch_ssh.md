## nsc scratch ssh

Create a temporary scratch instance and SSH to it. The instance is destroyed on exit.

```
nsc scratch ssh [command] [flags]
```

### Options

```
  -T, --disable-pty             Disable pseudo-terminal allocation.
  -t, --force-pty               Force pseudo-terminal allocation.
  -h, --help                    help for ssh
      --machine_type string     Specify the machine type.
  -A, --ssh_agent               If specified, forwards the local SSH agent.
      --wait_timeout duration   For how long to wait until the instance becomes ready. (default 2m0s)
```
