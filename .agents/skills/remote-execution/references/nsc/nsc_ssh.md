## nsc ssh

Start an SSH session.

```
nsc ssh [instance-id] [command] [flags]
```

### Options

```
      --container_name string   The name of the container to SSH to. If no container is specified, ssh to the instance environment.
  -T, --disable-pty             Disable pseudo-terminal allocation.
  -t, --force-pty               Force pseudo-terminal allocation.
  -h, --help                    help for ssh
      --oneshot                 If specified, a temporary instance will be created and destroyed upon disconnection.
  -A, --ssh_agent               If specified, forwards the local SSH agent.
      --unique_tag string       If specified, creates a instance with the specified unique tag.
```
